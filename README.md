Mystery Word Guessing Game

Overview :

Mystery Word Guessing Game is a simple browser-based game where the computer selects a random word from a random category, and the player must guess it one letter at a time. The game is inspired by classic word-guessing games and is a fun way to practice logic and vocabulary.

Features :

1. Randomly selects a category and word for each round.
2. Allows guessing letters one at a time.
3. Tracks guessed letters and remaining attempts.
4. Alerts player on correct or wrong guesses.
5. Ends game with a win or loss message.
6. User-friendly prompts and alerts for interaction.

How to Play : 

1. A random category and word are chosen by the computer.
2. You have 10 tries to guess the secret word.
3. Enter one letter at a time (a-z).
4. Correct guesses reveal the letters in the word.
5. Wrong guesses reduce your remaining tries.
6. The game ends when you either guess all letters correctly or run out of tries.
